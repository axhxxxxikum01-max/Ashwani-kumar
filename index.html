How it works

Each profile gets a unique account ID. Transfers are simulated locally (localStorage). Use the Receive QR to share your account id (and optional amount). Scan QR to auto-fill recipient.
Note: This demo stores passwords locally in the browser for convenience only — do not use real passwords or real money.
Download the file (Save As .html) and open in a modern browser. This is a local demo — no server, no real money. For multiplayer across devices you would need a backend (not included).
